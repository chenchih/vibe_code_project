    Description

      A user configured the 2.4GHz Wi-Fi SSID to use WPA3/WPA2 Mixed Mode. After switching the security setting back to WPA2-PSK (AES) only, two separate clients connected to the same SSID (one mobile, one laptop) can still ping each other and access files via an open shared folder, despite the "Client Isolation" setting being visibly enabled in the WebGUI.
